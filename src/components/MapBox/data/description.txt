Точки (marks):
lat - C.Ш.
lon - В.Д.

Области (Polygons):
position - массив точек полигона, первое значение - lat, второе - lon,


popup - открывается при клике на точку или полигон

tooltip - открывается при наведении на точку или полигон:
	-sticky - если true то окно при наведении будет отображаться там где курсор(есть только у тултипа полигона)

	-permanent - если true то будет виден всегда без необходимости наводиться (только у тултипа)

title - то что будет отображаться вверху окна
description - описание, для переноса на новую строку использовать \n, например:
	Лес большой -> Лес большой
	
	Лес \nбольшой -> Лес
			 большой

	Лес \n большой-> Лес
			  большой

(после \n все пробелы учитываются)

Для разбивки на слои нужно данные каждого слоя сохранять в разные файлы
